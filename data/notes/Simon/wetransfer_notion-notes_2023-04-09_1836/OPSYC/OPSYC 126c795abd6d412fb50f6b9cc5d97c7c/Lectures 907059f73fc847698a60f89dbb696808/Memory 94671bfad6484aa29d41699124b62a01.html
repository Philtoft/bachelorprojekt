Page Hit

Reference to VM word that is in physical memory (DRAM cache hit).

Page Fault

Reference to VM word that is not in physical memory (DRAM cache miss).

Handling Page Fault

Page miss causes page fault (an exception). Page fault handler selects a victim to be evicted (here VP 4).

Shared Memory in Linux

Linux default: no sharing between processes. If you want sharing (shared memory), then yo have to work for it. There are primitives in the C std lib for this.
