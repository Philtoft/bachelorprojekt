Proofs and Judgements (Conjunction, Implication)

The units of logics are propositions and judgements (1)

  • A proposition is a sentence that can have a truth value: either true (T) of false (F).
    • E.g. a triangle has three sides (T)
  • They can depend on context and compound — and some sentences are not propositions.
  • A judgement is the fundamental notion of assigning truth to a proposition, which can be based on evidence.

Disjunction (A∨BA∨ BA∨B﻿) is introduced by the rules ∨I1∨I_1∨I1​﻿ and ∨I2∨I_2∨I2​﻿, and eliminated by the rule ∨Eu,v∨E^{u,v}∨Eu,v﻿: to prove CCC﻿ true under the assumption A∨BA∨ BA∨B﻿ true we need to have a proof of CCC﻿ true assuming A true‾u\overline{A{\thinspace}true}^uAtrueu﻿ and a proof of CCC﻿ true assuming B true‾u\overline{B{\thinspace}true}^uBtrueu﻿.
      • Disjunction introduction rules:

        Here are the two rules for the disjunction introduction, and we read them from bottom to top.

        In order to conclude that A or B is true we need to proof that A is true, or otherwise to proof that B is true.

      • Disjunction elimination rule:

        If we want to eliminate this junction, we use the elimination rule. We read it from the left.

        In case we know that A or B is true, then you can conclude that some C is true if you can proof that A implies C, and separately (note the different u and v) you can proof that from the assumption of B you can conclude C.

    1. True and false are special:
      • True (T) only has an introduction rule (i.e. T is always true):

        It has no elimination rule because we cannot in any way conclude that true false.

      • False (F) only has an elimination rule (ex falso quodlibet, which is Latin for from false anything follows):

        It has no introduction rule; there is no way we can conclude that false is true.

        If we reach false in some case, then we can conclude that anything is possible.

    1. To introduce negation (¬A¬A¬A﻿ in the book ∼A^∼A∼A﻿, read as not AAA﻿) we assume A true‾u\overline{A{\thinspace}true}^uAtrueu﻿ and try to derive a contradiction, that is any proposition ppp﻿.
      • Negation introduction rule

        If you want to conclude that not A is true, you can start from a assumption that A is true (labeled by u), and then conclude that false is true. Then A is false.

      • Negation elimination rule

        If you reach a derivation (a judgement) that allows you to conclude that not A is true and A is true, you have reached a contradiction, and therefore you can conclude that any other fact C is true.

    1. Classical Logic.
      • Law of Exclude Middle:

        For any preposition it is always true that the particular preposition A (in this case) is true or not A is true.

      • Proof by contradiction (FCuF^u_CFCu​﻿):

        It allows you to conclude that A is true, starting from the hypothesis that not A is true if you reach a contradiction there.

      • Double negation elimination (¬¬C¬¬C¬¬C﻿):

        It allows you to conclude that A is true if you can conclude that not-not A is true.
